üì± Device Overview

PiTrezor is a hardware wallet running on a Raspberry Pi Zero W with an ILI9341 color display. It provides secure cryptocurrency storage and transaction management with an intuitive touchscreen interface.

Device Components

üñ•Ô∏è Color Display

3.5-inch ILI9341 display showing wallet status, transactions, and seed phrases

üîò Control Buttons

YES button (confirm) and NO button (cancel) for user input and menu navigation

üì° WiFi Connection

Built-in WiFi for Trezor Suite connectivity without USB cables

üîê Secure Processor

Dedicated Trezor firmware runs on isolated system for maximum security

Supported Cryptocurrencies

PiTrezor supports hundreds of cryptocurrencies including:

  • Bitcoin (BTC) - The original cryptocurrency
  • Ethereum (ETH) - Smart contract platform
  • Litecoin (LTC) - Bitcoin fork with faster blocks
  • Ripple (XRP) - Payment protocol
  • Dogecoin (DOGE) - Community coin
  • And 100+ more - Check Trezor Suite for full list

üöÄ Getting Started

First Power-On

Initial Setup

  1. Connect 5V USB power to the Pi Zero W
  2. Green LED will blink (indicates normal operation)
  3. Color display will show Trezor boot animation (1-2 minutes)
  4. Display shows "Welcome to Trezor" message
  5. Device is ready for initial configuration

WiFi Setup

Connecting to Your Network

  1. Press YES button on the welcome screen
  2. Navigate to "WiFi Settings" using YES/NO buttons
  3. Device scans for available networks
  4. Select your network from the list (YES = select, NO = previous)
  5. Enter WiFi password using on-screen keyboard
  6. Press YES to connect
  7. Device will display connected IP address
üí° Tip: The IP address shown on the display is needed to connect with Trezor Suite on your computer.

Connecting to Trezor Suite

Remote Wallet Connection

  1. Download Trezor Suite on your computer
  2. Open Trezor Suite application
  3. Click "Add Device" or "Connect Device"
  4. Select "Enter IP address" for network connection
  5. Enter the IP address shown on PiTrezor display
  6. Set port: 21325 (default)
  7. Click "Connect"
  8. Authorize connection on PiTrezor display (press YES)
  9. Trezor Suite is now connected to your device
‚ö†Ô∏è Security Note: Only enter Trezor Suite PIN and passphrase on the PiTrezor device itself, never on your computer. This prevents keylogging attacks.

üîê Creating a Wallet

‚ö†Ô∏è Critical Security Warning: Your seed phrase is the only backup for your wallet. Write it down on paper, store it securely, and NEVER show it to anyone or enter it into a computer. Anyone with your seed phrase can access all your funds.

Generate New Wallet

Creating Your Recovery Seed

  1. On home screen, press YES to enter menu
  2. Navigate to "Settings" using YES/NO buttons
  3. Select "Initialize Device"
  4. Choose "Generate New Seed"
  5. Select seed length: 12 words (recommended) or 24 words (maximum security)
  6. Device generates seed phrase
  7. Write down all words in order on paper
  8. Device will confirm you wrote down the seed (test random words)
  9. Set a PIN (4-8 digits) for device protection
  10. Create optional passphrase for additional security
  11. Wallet is now initialized and ready to use

Seed Phrase Safety

‚úÖ DO:
  • Write seed phrase on paper with pen
  • Store in a safe, secure location
  • Keep multiple copies in different locations
  • Consider waterproof/fireproof storage
  • Memorize the seed (as backup to written copy)
‚ùå DON'T:
  • Never type seed into a computer
  • Never take a photograph of the seed
  • Never share seed with anyone
  • Never store digitally (email, cloud, etc.)
  • Never leave written seed unattended

PIN Protection

Your PIN protects the device from unauthorized access. Choose a strong PIN:

  • Minimum: 4 digits
  • Recommended: 8+ digits
  • Avoid: Sequential numbers (1234) or repeating digits (1111)
  • Security: Device erases after 3 failed attempts

üì§ Sending Cryptocurrency

Prepare Your Transaction

Using Trezor Suite (Recommended)

  1. Open Trezor Suite on your computer
  2. Select the cryptocurrency you want to send
  3. Click "Send" button
  4. Enter recipient address in the address field
  5. Enter amount to send
  6. Set transaction fee (fast/normal/slow)
  7. Review transaction details
  8. Click "Send" - this sends request to PiTrezor

Confirm on PiTrezor Device

On-Device Confirmation

  1. PiTrezor display shows transaction details:
    • Receiving address
    • Amount being sent
    • Transaction fee
    • Total cost
  2. Carefully review all details match what you intended
  3. Verify recipient address matches (especially first and last characters)
  4. If correct, press YES button to confirm
  5. If wrong, press NO button to cancel
  6. Device signs transaction and broadcasts it
  7. Display shows "Transaction Sent" confirmation
  8. Trezor Suite shows confirmation on computer
‚ö†Ô∏è Important: Once you confirm a transaction on PiTrezor, it cannot be cancelled. Always triple-check the recipient address and amount before pressing YES.

Transaction States

Status Meaning Action Needed
Pending Transaction submitted to blockchain Wait for confirmation
Confirming (1/X) Network is processing transaction Wait for more confirmations
Confirmed Transaction is secure and final None - transaction complete
Failed Transaction was rejected by network Check error message and retry

üì• Receiving Cryptocurrency

Generate Receive Address

Create Address for Receiving Funds

  1. On home screen, press YES to open menu
  2. Navigate to "Accounts"
  3. Select the cryptocurrency you want to receive
  4. Select or create the account
  5. Press YES on "Generate Address" or "Show Address"
  6. Display shows QR code and address string

Share Address with Sender

You can share your receive address in these ways:

  • QR Code: Sender scans QR code with phone camera
  • Text Address: Copy and paste the address to sender
  • Display View: Show display to someone to scan
üí° Important: Each time you request a new address, PiTrezor generates a unique address. This is normal and adds to privacy. All addresses receive funds to the same wallet.

Verify Received Funds

  1. Wait for sender to confirm they sent funds
  2. Return to home screen - balance will update
  3. In Trezor Suite, you can see transaction details
  4. Transaction shows as "Pending" initially
  5. Wait for blockchain confirmations (varies by cryptocurrency)
  6. After sufficient confirmations, funds are secure

üîí Security Features

Pin Protection

Your PIN protects the device from physical theft. Features include:

Unlock Requirement

PIN required every time device boots or wakes from sleep

Attempt Limit

After 3 wrong PIN entries, device erases all data

No Backdoor

No way to recover device without PIN - not even manufacturers

On-Device Entry

PIN entered only on device display, never on computer

Passphrase (Advanced)

Optional additional security layer above PIN:

  • What it does: Adds extra words to your seed phrase
  • Effect: Creates completely different wallet with same seed
  • Security: Protects against seed theft (thief gets wrong wallet)
  • Risk: If you forget passphrase, you can't recover wallet
  • Use case: Advanced users with multiple wallets

Address Verification

PiTrezor verifies addresses before signing transactions:

  • Device confirms each receive address is yours
  • Display shows address for you to verify
  • Protects against malware redirecting funds
  • Always verify address matches between devices

Air-Gapped Security

Key security principle: Your private keys never leave the device

  • Private keys stored only in secure Trezor firmware
  • Keys never sent to computer or network
  • Only transaction approvals sent to network
  • Effectively air-gapped while using WiFi (keys stay local)

Recovery Scenarios

Scenario Recovery Method
Forgot PIN Device erases. Use seed phrase to recover on new device.
Lost Device Use seed phrase to restore wallet on new Trezor device.
Stolen Device PIN protects it. Move funds using new device + seed phrase.
Corrupted System Reflash firmware. Use seed phrase to restore wallet.

üìã Common Tasks

Check Your Balance

View Account Balance

  1. Turn on device and unlock with PIN
  2. Home screen shows wallet balance (if WiFi synced)
  3. Press YES and go to "Accounts" for detailed breakdown
  4. Select cryptocurrency to see balance in that currency
  5. Press YES to see transaction history

Update Firmware

Keep Device Secure with Latest Firmware

  1. Connect PiTrezor to WiFi
  2. Open Trezor Suite on your computer
  3. Trezor Suite checks for firmware updates
  4. If update available, click "Update Firmware"
  5. Follow on-screen instructions on both devices
  6. Device will restart after update completes
  7. Unlock with PIN to confirm update success

Backup Seed Phrase

Create Written Backup

  1. Go to Settings menu (YES from home)
  2. Select "Backup Device"
  3. Device displays your recovery seed
  4. Write down ALL words in exact order on paper
  5. Include numbers next to each word (1/12, 2/12, etc.)
  6. Verify you wrote them correctly
  7. Store in secure location

Change PIN

Update Your Device PIN

  1. Go to Settings menu
  2. Select "Change PIN"
  3. Enter current PIN to confirm
  4. Enter new PIN using on-screen keyboard
  5. Confirm new PIN by entering it again
  6. Press YES to save
  7. Next boot will require new PIN

Manage Multiple Accounts

Create separate accounts for different purposes:

  1. Go to Accounts menu
  2. Select cryptocurrency
  3. Press YES on "New Account"
  4. Name the account (e.g., "Savings", "Daily Use")
  5. Each account has separate addresses and balance
  6. All accounts backed by same seed phrase

View Transaction History

  1. Go to Accounts menu
  2. Select the account to view
  3. Press YES on "Transactions"
  4. Scroll through past transactions with YES/NO
  5. Press YES on any transaction to see details
  6. Shows: amount, date, sender/receiver, status

üêõ Troubleshooting

Device Won't Turn On

No Display or LED Activity

  1. Check power connection to USB port
  2. Try different USB cable (current one may be broken)
  3. Try different power adapter (should be 5V 2A minimum)
  4. Check if USB port is dirty - clean with dry cloth
  5. If LED still doesn't blink, device may need factory reset

Forgot PIN

Device Locked by PIN

  1. After 3 wrong PIN attempts, device erases
  2. Recover wallet using your seed phrase on new device
  3. To recover on PiTrezor:
    • Choose "Restore from Seed" on new device
    • Enter your 12 or 24 word seed phrase
    • Set new PIN
    • Wallet restored with all previous addresses

WiFi Won't Connect

Network Connection Issues

  1. Check WiFi SSID is visible and broadcasted
  2. Verify WiFi password is correct (case-sensitive)
  3. Try connecting to 2.4GHz network (device doesn't support 5GHz)
  4. Move closer to WiFi router
  5. Check router isn't blocking device MAC address
  6. Restart router by unplugging for 30 seconds
  7. Redo WiFi setup on device: Settings > WiFi Configuration

Transaction Won't Send

Trezor Suite Shows Error

  1. Check device is unlocked (PIN entered)
  2. Verify address is valid and correct format
  3. Check account has sufficient balance for amount + fee
  4. Try again with lower fee (slower but cheaper)
  5. Restart Trezor Suite
  6. Check if network connection is stable
  7. Try different cryptocurrency network

Display Showing Wrong Information

Balance or Transaction Incorrect

  1. Device balance syncs from blockchain (takes time)
  2. First time may take several minutes to sync
  3. Try manually refreshing in Trezor Suite
  4. Check if transaction is pending (not confirmed yet)
  5. Verify you're looking at correct account/cryptocurrency
  6. Restart device (unplug and replug USB)

Device Freezes or Unresponsive

Button Presses Not Responding

  1. Device may be processing (wait 30 seconds)
  2. Try pressing button again
  3. Unplug USB power and wait 10 seconds
  4. Reconnect USB power
  5. Device will reboot and show home screen
  6. Unlock with PIN
  7. If persistent, may need firmware reflash

Can't Connect to Trezor Suite

Remote Connection Issues

  1. Verify PiTrezor is connected to WiFi
    • Check IP address showing on display
    • If no IP shown, WiFi not connected
  2. On computer, verify network can reach Pi
    • Windows: ping [IP-ADDRESS] in Command Prompt
    • Mac/Linux: ping [IP-ADDRESS] in Terminal
  3. Verify port 21325 is open on router
  4. Restart Trezor Suite application
  5. Try connecting with new IP address entry
  6. Check firewall settings on computer

Always Remember:
  • Seed Phrase: Is the only way to recover your wallet. Write it down and store securely offline.
  • PIN: Protects your device. Choose strong PIN and don't share with anyone.
